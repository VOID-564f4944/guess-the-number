# Guess The Number 🎯

A simple C++ console-based game where the player tries to guess a randomly generated number within a limited number of attempts based on the selected difficulty.

---

## 📌 Features

* Three difficulty levels:

  * Easy (10 attempts)
  * Medium (7 attempts)
  * Hard (5 attempts)
* Random number generation using modern C++ (`<random>`)
* Input-based gameplay loop
* Feedback system:

  * "Too high"
  * "Too low"
  * "Correct"

---

## 🛠️ Technologies Used

* C++
* Standard Library (`<iostream>`, `<random>`)
* Visual Studio
* Git + GitHub

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/VOID-564f4944/guess-the-number.git
   ```

2. Open the project in Visual Studio

3. Build and run the program

---

## 🎮 How to Play

1. Choose a difficulty level
2. The program generates a number between 1 and 100
3. Enter your guesses
4. You will be told if your guess is too high or too low
5. Win by guessing correctly within the allowed attempts
